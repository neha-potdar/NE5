RTL Version

  1. Docs
  2. RTL Version

In order to have RTL mode enabled, replace the reference of bootstrap.min.css to bootstrap-rtl.min.css and app.min.css to app-rtl.min.css in the views/Partials/HeaderStyle.ejs and add dir="rtl" in html tag in file views/layout.ejs and comment out function initSettings in the file public/js/app.js.

Also we have added jquery to make default mode to light so you must have to change in function initSettings in the file public/assets/js/app.js has
sessionStorage.setItem("is_visited", "light-mode-switch");
change above line's "light-mode-switch" to "rtl-mode-switch" as per below line
sessionStorage.setItem("is_visited", "rtl-mode-switch"); to make default RTL version.
