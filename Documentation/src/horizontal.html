You may copy paste views/layoutsHorizontal.ejs code into the views/layout.ejs for set default as horizontal menu.

How to use pre-built layouts?

Each of the layout options is provided below with steps you would need to perform in views/layout.ejs in the body object:

Topbar Dark
Keep your body element with data attribute data-topbar="dark" data-layout="horizontal" E.g. <body data-topbar="dark" data-layout="horizontal"> to have dark topbar.
Topbar Light
Keep your body element with data attribute data-layout="horizontal" E.g. <body data-layout="horizontal"> and remove data-topbar element to have light topbar.
Boxed Layout
Keep your body element with data attribute data-layout-size="boxed" data-layout="horizontal" E.g. <body data-layout-size="boxed" data-layout="horizontal"> to have boxed layout.
Preloader
Keep your body element with data attribute data-topbar="dark" data-layout="horizontal" E.g. <body data-topbar="dark" data-layout="horizontal" > to have scrollable layout. Add preloader HTML code in body tag: <div id="preloader"> <div id="status"> <div class="spinner-chase"> <i class="chase-dot"></i> <i class="chase-dot"></i> <i class="chase-dot"></i> <i class="chase-dot"></i> <i class="chase-dot"></i> <i class="chase-dot"></i> <i class="chase-dot"></i> </div> </div> </div>
